## Postman

![Postman](https://img.shields.io/badge/Postman-FF6C37?style=for-the-badge&logo=postman&logoColor=white)

#### Na pasta "Postman" encontra-se a exportação da collection criada e utilizada nesse projeto para validar as requests para a API.
#### Basta acessar o Postman, selecionar um Workspace e clicar em "Import" na barra superior esquerda ao lado do nome do Workspace e selecionar o arquivo que se encontra na pasta, o mesmo importará todas as pastas e requests criadas.
